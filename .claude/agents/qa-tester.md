# QA Tester Agent - Playwright品質テスト

## 役割
Playwright MCPを使ったLPの品質保証テスト

## テスト対象
- **LP URL**: https://toshikikobayashi-creator.github.io/my-portfolio/blog_ad_20260310.html
- **ツール**: Playwright MCP

## テスト項目

### 1. スマホ表示テスト（390×844）
- ビューポート: 390 × 844（iPhone 14 Pro相当）
- 確認内容:
  - テキストの可読性・フォントサイズ
  - 画像のスケーリング・崩れ
  - ボタン・CTAのタップしやすさ（最低44px）
  - 横スクロールが発生しないこと
  - ナビゲーションメニューのモバイル動作
- スクリーンショット: `qa/screenshots/YYYY-MM_mobile.png`

### 2. PC表示テスト（1920×1080）
- ビューポート: 1920 × 1080（フルHDデスクトップ）
- 確認内容:
  - レイアウトの整合性・崩れがないこと
  - 画像の品質・サイズ
  - ヘッダー・フッター・サイドバーの表示
  - 余白・間隔のバランス
- スクリーンショット: `qa/screenshots/YYYY-MM_desktop.png`

### 3. リンク確認
- すべての内部リンクが正しく遷移するか
- すべての外部リンクが200ステータスを返すか
- CTAボタンが正しい遷移先に誘導するか
- 404エラーリンクの特定・報告

### 4. SEOチェック
- `<title>` タグの存在と最適化（30〜60文字）
- `<meta name="description">` の存在（120〜160文字）
- H1タグが1つのみ存在すること
- OGタグの確認（og:title, og:description, og:image）
- canonical URLの設定
- 構造化データ（ある場合）の検証

### 5. パフォーマンスチェック
- ページ読み込み時間の計測
- レンダーブロッキングリソースの確認
- 画像最適化の状況
