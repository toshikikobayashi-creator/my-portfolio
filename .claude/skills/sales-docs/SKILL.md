## ROI計算式

### 基本指標定義

```
【投入コスト】
広告費用 = プラン基本料 + オプション合計（税抜）

【リーチ計算】
1時間あたりの推定通行人数（エリア別参考値）
  - 渋谷ハチ公前・センター街: 約20,000〜30,000人/時間
  - 新宿東口・歌舞伎町: 約15,000〜25,000人/時間
  - 原宿竹下通り: 約10,000〜20,000人/時間
  - 銀座中央通り: 約8,000〜15,000人/時間
  - 秋葉原電気街: 約8,000〜12,000人/時間

視認率（推定）: 15〜25%（通行人のうちディスプレイを目視した割合）
記憶率（推定）: 20〜30%（視認者のうち広告内容を記憶した割合）
```

### ROI計算フォーミュラ

```
■ 推定総リーチ数（ユニーク接触者）
総リーチ数 = 1時間あたり通行人数 × 配信時間 × 視認率

例: 渋谷・4時間・1名配信の場合
= 25,000人 × 4時間 × 0.20 = 20,000人/日

■ 有効接触数（記憶に残った人数）
有効接触数 = 総リーチ数 × 記憶率
= 20,000人 × 0.25 = 5,000人/日

■ CPM（1,000インプレッションあたりコスト）
CPM = (広告費用 / 総リーチ数) × 1,000
例: スタンダードプラン ¥25,000 / 20,000人 × 1,000 = ¥1,250

【参考】他媒体のCPM相場
  - アドトラック:    ¥8,000〜¥15,000
  - 電車内交通広告:  ¥3,000〜¥8,000
  - Web広告（ディスプレイ）: ¥500〜¥3,000
  - ヒューマンアド:  ¥800〜¥2,000（競争力ある水準）

■ コンバージョン推定（店舗集客・来店促進の場合）
推定来店者数 = 有効接触数 × 来店転換率（0.5〜2%）
例: 5,000人 × 1% = 50人/日

売上効果（推定）= 推定来店者数 × 購買率 × 客単価
例: 50人 × 30% × ¥3,000 = ¥45,000/日

■ ROI（投資対効果）
ROI = (売上効果 - 広告費用) / 広告費用 × 100%
例: (¥45,000 - ¥25,000) / ¥25,000 × 100% = 80%

■ SNS二次拡散の付加価値（参考）
SNSインプレッション推定 = (撮影者数) × (フォロワー平均数) × 拡散係数
  ※ 撮影者数 = 総リーチ数 × SNS投稿率（0.1〜0.5%）
  ※ フォロワー平均 = 500〜2,000人（一般ユーザー想定）
  ※ 拡散係数 = 1.5〜3.0（リポスト・いいねによる2次拡散）

例: 20,000人 × 0.3% × 1,000人 × 2.0 = 120,000インプレッション/日（付加価値）
```

## 使い方（スキル呼び出し方法）

このスキルは以下のユーザー発話で起動します:
- 「営業提案書を作って」
- 「[クライアント名]向けに見積書を作成して」
- 「ROIをシミュレーションして」
- 「ヒューマンアドの提案資料を作って」

### 出力先ファイルパス
```
my-portfolio/
└── sales/
    ├── proposals/    # 提案書 (proposal_クライアント名_日付.md)
    ├── quotes/       # 見積書 (quote_クライアント名_日付.md)
    └── reports/      # 効果測定レポート (report_クライアント名_日付.md)
```

### 作成フロー
1. クライアント情報・目的・予算・希望エリアをヒアリング
2. 最適プランを提案書テンプレートに当てはめて作成
3. 見積書を別ファイルで作成
4. ROIシミュレーション表を見積書に添付
5. `sales/` ディレクトリに保存して報告
