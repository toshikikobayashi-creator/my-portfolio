# カテゴリ別ペインポイント辞書

企業が最も「恐れている」ことをカテゴリ別に整理。
文章生成時、該当カテゴリの恐れを1-2個選び、②パートで使用する。

---

## 新曲・音楽

### レコード会社・レーベル向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 新曲を出しても埋もれる | 「毎週数百曲がリリースされる中で、街で"偶然出会う"体験がストリーミング再生を動かします」 |
| デジタル広告のCPAが高騰 | 「SNS広告のCPM上昇が止まらない中、1人あたり○円で"実際に目にする"接点を作れます」 |
| ファンの熱量が可視化できない | 「街頭での反応・写真撮影・QRタップ数で、リアルな熱量をデータ化できます」 |

### 音楽フェス・イベント主催向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 当日の来場数が読めない | 「会場周辺で"今やってる感"を演出し、通りがかりの来場を生み出します」 |
| チケットが売れ残る | 「開催直前の街頭プロモーションで"ラストチャンス"の空気を作ります」 |
| 近くにいるのに存在を知らない | 「半径500mの"知らなかった層"にピンポイントで届けます」 |

## 新商品ローンチ

### 化粧品・コスメブランド向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 発売しても店頭で手に取ってもらえない | 「"今日出たばかり"という体験を街頭で作り、その足でお店に向かわせます」 |
| サンプリングのコスパが悪い | 「ターゲットエリアを絞ったサンプリングで、1個あたりの配布コストを最適化します」 |
| デジタルだけでは質感が伝わらない | 「実物を手に取る体験 × QRでECへ誘導、オフラインとオンラインを繋ぎます」 |

### 飲料メーカー向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 新商品の棚取り競争が激しい | 「発売初週に"街で話題になっている"空気を作り、店頭の回転率を上げます」 |
| 試飲イベントのコストが高い | 「固定ブース不要。歩きながら配れるから、設営費ゼロで即日スタート」 |
| 数量限定品が話題にならない | 「"今しか手に入らない"を街頭で演出し、SNS投稿と来店を同時に誘発します」 |

### 飲食・新規オープン向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| オープンしても知られない | 「開店初日から半径1km圏内に"今日オープン"を届けます」 |
| チラシの反応率が下がり続けている | 「紙のチラシは0.1%以下。動くLEDディスプレイなら目に入らないほうが難しい」 |
| 広告費をかけても来店につながらない | 「お店の目の前を歩く人に直接アプローチ。"来店まで徒歩3分"の広告です」 |

### 小売・旗艦店オープン向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| オープン初日の行列が作れない | 「周辺エリアを歩いて"今オープン"を知らせ、初日の導線を作ります」 |
| Webだけでは"わざわざ行く理由"が弱い | 「街頭で限定特典やオープン記念を告知し、"今行かなきゃ"を創出します」 |

## イベント・POPUP

### エンタメIP（アニメ・ゲーム・VTuber）向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| ファンにしか届かない | 「POP-UP周辺の"まだ知らない層"に存在を知らせ、新規ファン獲得の接点を作ります」 |
| SNS告知だけでは来場につながらない | 「"今やってる、すぐそこで"というリアルタイムの導線が、SNSにはない行動喚起を生みます」 |
| 物販の売上が会場キャパに依存する | 「会場外から人を呼び込むことで、入場者数の上限を超えた売上機会を作ります」 |

### キャラクターIP・ファミリー向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 子ども連れの来場導線が弱い | 「親子が歩くエリアをピンポイントで回り、"あ、あれ見たい！"を引き出します」 |
| コラボPOP-UPの認知が弱い | 「コラボ先のファン層 × 通りがかり層の両方にリーチします」 |

### 展示会・体験型イベント向け
| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| すぐ近くでやっているのに誰も来ない | 「会場から半径500mの"存在を知らない人"に歩いて届けます」 |
| 期間限定なのに認知が広がる前に終わる | 「初日から周辺の人の目に入る。開催期間をフルに活かせます」 |
| チケットの当日券が動かない | 「"今からでも入れます"を会場周辺でリアルタイム告知します」 |

## 新規事業・スタートアップ

| 恐れ | 刺さるフレーズ例 |
|------|------------------|
| 革新的な技術なのに一般認知が低い | 「展示会の外でも"知ってもらう接点"を作り、toBだけでなくtoC認知も広げます」 |
| 展示会以外でリーチする手段がない | 「ブース来場を待つのではなく、会場周辺を歩いてブースに"連れてくる"動線を作ります」 |
| 広報予算が限られている | 「設備投資ゼロ・最短即日。限られた予算で最大の"目に入る"効果を出します」 |
| 資金調達後の認知獲得が課題 | 「プロダクトの"存在を知ってもらう"最初の一歩を、街頭から始められます」 |

## ペインポイント選定ルール

1. **リストの「詳細」「提案ポイント」カラムを最優先で参照** → 企業固有の文脈に合うものを選ぶ
2. **カテゴリの恐れから最もフィットするものを1-2個選択**
3. **企業Webリサーチで得た情報と組み合わせてカスタマイズ**
4. **同じカテゴリでも企業ごとに異なる恐れを使い分ける**（使い回し厳禁）
5. フレーズ例はそのまま使わず、企業の具体的状況に合わせて必ずアレンジすること
