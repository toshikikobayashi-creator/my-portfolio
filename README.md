# 俊輝のポートフォリオ

## 概要

俊輝（Toshiki）による個人ポートフォリオプロジェクトです。自己紹介ページ、ToDoリストアプリ、売上データ集計ツールの3つのWebアプリケーションで構成されています。すべてのアプリケーションは単一のHTMLファイルで完結しており、サーバー不要でブラウザだけで動作します。デザインはグリーン系のグラデーションで統一されたモダンなUIとなっています。

## ファイル一覧

| ファイル名 | 説明 |
|---|---|
| `index.html` | **自己紹介ページ** - 俊輝のプロフィールカード。名前、自己紹介文、趣味・興味（AI学習、Webサイト制作、自動化、データ分析、ビジネス構築）をカード形式で表示する。 |
| `todo.html` | **ToDoリストアプリ** - タスク管理アプリケーション。タスクの追加・完了・削除が可能。締切日の設定に対応し、期限切れタスクを赤色で警告表示する。データはlocalStorageに保存されるため、ブラウザを閉じても内容が保持される。 |
| `sales.html` | **売上データ集計ツール** - CSVファイルをドラッグ＆ドロップまたはファイル選択で読み込み、商品別の売上を自動集計する。集計結果をテーブルと横棒グラフで表示し、CSVダウンロード機能も備える。サンプルデータでの動作確認も可能。 |
| `CLAUDE.md` | **プロジェクトルール** - 開発時のルール定義ファイル（コメントは日本語、変更前の説明、変更後の報告、モダンなデザイン）。 |

## 使い方

1. 各HTMLファイルをブラウザで直接開くだけで動作します。サーバーの構築やインストール作業は不要です。
2. `index.html` - ブラウザで開くとプロフィールカードが表示されます。
3. `todo.html` - テキスト欄にタスクを入力し「追加」ボタンまたはEnterキーで追加できます。締切日の設定も可能です。チェックボックスで完了、バツボタンで削除できます。
4. `sales.html` - CSVファイル（形式: 日付, 商品名, 数量, 単価）をドロップエリアにドラッグ＆ドロップするか、「サンプルデータで試す」ボタンで動作を確認できます。集計結果はCSVとしてダウンロードすることも可能です。

## 使用技術

- **HTML5** - ページ構造
- **CSS3** - グラデーション、Flexbox、レスポンシブデザイン、アニメーション
- **JavaScript** - DOM操作、イベント処理、データ集計ロジック
- **localStorage** - ToDoリストのデータ永続化
- **Canvas API** - 売上グラフの描画
- **File API / Drag & Drop API** - CSVファイルの読み込み
- **Blob API** - CSV集計結果のダウンロード
